Janela de impressão

 

 

A impressão dos registros de uma consulta agora exibe uma janela de impressão com mais opções (figura 1), tais como:

-          Barra superior

-          Exibe o total de obras selecionadas para a impressão atual (no exemplo “2285”) e número de obras com imagens existentes na seleção (no exemplo “842”).

-           Exemplo do arquivo a ser impresso tomando como base uma das obras selecionadas para impressão.

-           Visualização do arquivo a ser impresso ou salvo.

NOTA: se a quantidade de obras com imagens que serão impressas for muito grande, um erro de impressão poderá ocorrer. Para verificar se o sistema conseguiu carregar todos os registros com suas respectivas imagens, é necessário ir até o final do arquivo para verificação. O limite de tamanho de arquivo para impressão dependerá da configuração de seu equipamento.

-          Fecha a janela de opções de impressão e retorna a consulta.

-           Visualização do arquivo a ser impresso ou salvo. Opção disponível apenas para impressão de etiquetas.

-           Salvar arquivo no formato Microsoft Excel. Algumas instituições utilizam o Excel para formatar os arquivos para impressão de etiquetas. Opção disponível apenas para impressão de etiquetas.

Nota: esta opção é utilizada normalmente para etiquetas de exposições e raramente o número de obras de uma exposição supera as duzentas obras estabelecidas na opção Etapas. Se este número for muito grande um erro de impressão poderá ocorrer. Isto dependerá da capacidade de seu equipamento.

-          Configurar página:

þ com cabeçalho – exibe ou oculta o cabeçalho do Simba no topo da página

þ com detalhes da seleção – exibe ou oculta as opções da consulta realizada. No exemplo a consulta realizada foi: Coleção: Pintura Brasileira. Este será o detalhe da seleção que será impresso.

þ com imagem – exibe ou oculta as imagens das obras da seleção que tiverem imagem associadas a elas. Para as obras que não possuem imagens será impresso o texto “Imagem não disponível” na área correspondente às imagens.

þ somente com imagem e obra tridimensional – opção normalmente utilizada para impressão de etiquetas onde apenas as imagens de obras tridimensionais que tiverem imagens associadas à elas serão impressas. As imagens das obras bidimensionais não serão impressas.

þ com tradução para o inglês – exibe ou oculta o texto traduzido para o inglês. Normalmente utilizado em etiquetas. Exemplo:

Paisagem , 1961
óleo sobre tela, 97 x 130 cm
assinada Jayme Aguiar 1961
compra, Jaime Aguiar, 1961

 

Landscape, 1961
óleo sobre tela, 97 x 130 cm
signed Jayme Aguiar 1961
acquisition, Jaime Aguiar, 1961

 

Nota: se o campo Título em inglês (Landscape) da ficha técnica não estiver preenchido, o sistema imprimirá o título como aparece no campo Título para etiqueta (Paisagem). O campo material/técnica não é traduzido automaticamente

 

þ etiqueta – opção utilizada para impressão de etiquetas.

Nota: quando a opção é selecionada, o sistema inibe automaticamente as opções “com cabeçalho” e “com detalhes da seleção”. Os ícones na barra superior são substituídos nesta opção (ver Barra Superior acima).

 

-          fonte: tamanho de fonte para impressão.

-          autor – tamanho da fonte que será utilizada para imprimir o nome do autor. Valor padrão = 18

-          texto – tamanho da fonte para o restante do texto de impressão. Valor padrão = 13

 

-          Configurar impressão:

Etapas: dependendo da quantidade de obras que serão impressas, do número de obras com imagem e do tamanho que estas imagens ocupam em disco, o computador poderá exibir uma mensagem de erro por não conseguir carregar todo o arquivo para exibição e/ou impressão. Se isto ocorrer, imprima a consulta por etapas, isto é, de 200 em 200 obras.

Nota: o valor padrão para quebra de impressão (200) pode ser mudado pelo administrador do Donato (ver Definindo valor da quebra de impressão abaixo).

Definindo valor da quebra de impressão

Para definir o número de obras que serão impressas por etapa, o administrador do Donato deverá seguir os seguintes passos:

1.      Abrir o item do menu Estrutura->Sistema->Parâmetros

2.      Selecionar o ícone  para alterar a opção ETAPAS_IMPRESSAO

3.      Alterar os campos Valor e Valor default para o valor desejado. O valor padrão é 200.

Nota: aumentar muito o valor para as etapas de impressão pode inviabilizar as impressões se seu computador não conseguir carregar todo o arquivo para impressão. O tamanho limite para arquivo depende da configuração do computador, além do tamanho e quantidade de imagens que serão impressas.
