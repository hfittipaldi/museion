Função Acervo/Movimentação/Incluir

 

Esta função permite fazer a movimentação de obras do acervo do museu para locais internos ou externos ou para exposições.  As exposições também podem ser internas ou externas.

 

 

No caso de uma movimentação para exposição interna informe o local de origem e de destino, além das datas de saída e retorno provável. Salve a movimentação clicando no botão

Gravar. Inclua as obras que participarão desta movimentação e complete a aba de exposições.

 

 

Na aba de Obras clique no botão de adicionar obras para poder agregar obras a esta movimentação:

 

 

Podemos procurar as obras pelo seu número de registro ou pelo seu título:

 

 

Pelo título, o sistema apresentará uma lista das obras que satisfaçam a condição de pesquisa. Clique no ícone de adicionar obra (página com o sinal de mais) para adicionar esta obra a movimentação:

 

 

 

A tela então ficará com esta aparência:

 

 

Já a adição por número de registro mostrará diretamente a obra selecionada. Clique no ícone de adicionar obra a movimentação para que ela seja agregada a esta movimentação:

 

 

 

Pronto. As obras estão agora agregadas a esta movimentação:

 

 

 

 

Vá para a aba Exposições e clique no ícone de adicionar exposições:

 

 

Caso seja uma nova exposição preencha todos os dados da exposição:

 

 

Se for uma exposição já existente, informe o título da exposição e o sistema devolverá uma lista de exposições para que seja selecionada a exposição desejada. Pressione o ícone à direita para adicionar a exposição à lista (fig. abaixo).

 

 

Agora já temos a exposição associada a nossa movimentação.
