Implantação do Sistema

 

A implantação do Projeto Donato 3.0 requer que alguns requisitos tenham sido previamente cumpridos, em particular com relação à instalação dos pacotes de software que servirão como alicerce da aplicação, a saber:

 

  • Sistema Operacional
  • Banco de Dados MySQL
  • Servidor Web Apache
  • Linguagem PHP

 

Estes componentes deverão estar disponibilizados, instalados e em funcionamento para que recebam as bases de dados e programas do projeto.

É importante lembrar que o servidor (CPU) onde estes pacotes de software serão executados já deverá estar disponível.

 

Além disto, as estações-clientes a serem utilizadas no projeto já deverão estar funcionando e com o navegador (browser) escolhido instalado e devidamente configurado para acesso ao servidor do Projeto Donato. Uma página de teste será provida e será apresentada se o servidor for acessado a partir da estação cliente e todas as configurações necessárias estiverem corretas. Somente a partir deste momento é que os programas poderão ser instalados e executados.

 

A instalação física (elétrica e rede) do servidor do Projeto Donato, a instalação do sistema operacional, sua configuração básica e configuração de rede, bem como a instalação, configuração e conexão das estações-clientes são de responsabilidade do MNBA, que deve prover toda a infra-estrutura necessária, como parte elétrica, cabeamento de rede, equipamentos de conexão (hub, switch) e demais recursos para que o projeto possa ser instalado.É importante verificar os requisitos descritos no documento “Componentes e Requisitos do Projeto Donato 3.0”

 

Este ambiente de servidores, estações-clientes, rede de dados e elétrica deverão estar disponibilizados para as etapas f e g do projeto, ou seja, deverão estar disponíveis até o início do mês de Maio de 2006.

 

Já a instalação e configuração dos componentes de software (MySQL, Apache, PHP) nas versões adequadas ao funcionamento do projeto no servidor (CPU) será de responsabilidade da Visão Virtual. As versões para estes produtos deverão estar de acordo com o documento “Componentes e Requisitos do Projeto Donato 3.0”.

 

 

Após a instalação dos componentes de software, será montado o banco de dados e realizada a migração da base de dados atualmente em ACCESS para o MySQL.

 

Uma vez portada a base de dados com os ajustes necessários para a correta entrada nas bases definidas no MySQL, será possível iniciar a etapa de execução do projeto e treinamento dos técnicos.

 

Nesta etapa, já deverá ter sido formado o elemento de ligação do MNBA, de forma que exista uma pessoa com capacitação no uso dos programas, e com capacidade de esclarecer dúvidas e propagar o conhecimento pela equipe de técnicos do MNBA, ou seja, o treinamento dos técnicos do MNBA e outras instituições que proventura venham a utilizar o sistema são de responsabilidade do MNBA.

 

A formação do técnico que será o elemento multiplicador da informação, responsável por propagar o treinamento e que também terá a capacitação para a instalação dos pacotes de software necessários, é de responsabilidade da Visão Virtual.

 

A manutenção do código, uma vez aprovado pelo MNBA, não é de responsabilidade da Visão Virtual, que foi contratada para o desenvolvimento da aplicação. É importante então que o técnico a ser designado para acompanhar o projeto esteja capacitado nas tecnologias envolvidas ou caso ainda não esteja familiarizado com o ambiente em questão que venha a faze-lo o mais brevemente possível.

 

A documentação da aplicação será on-line, ou seja, os campos são auto-explicativos com hints e mensagens de esclarecimento. Existe ainda uma versão completa do manual de utilização do sistema embutida no mesmo, de forma que pode-se a qualquer momento ter acesso a este material e esclarecer dúvidas quanto ao uso ou quanto algum recurso do sistema.
