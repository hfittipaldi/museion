Função Acervo/Obra/Incluir com Publicação

Ver também Incluir – Alterar – Alterar com Publicação - Registrar

 

Esta função permite incluir novas obras no acervo do museu com a opção de já poder publicar o registro.

Para isso clique nesta opção e informe na tela a seguir o número de registro da obra:

 

 

Uma vez que o Registro informado esteja livre o sistema perguntará se de fato se deseja incluir esta obra. Clique em OK:

 

 

Devido a quantidade de campos necessários para o cadastramento da obra, foi necessário fazer este formulário utilizando-se toda a área da tela. Note que ao lado do nome do usuário logado existe um ícone que permite retornar a tela com as opções de menu.

 

 

 

Preencha todos os campos necessários para o armazenamento da obra. Além das partes e dos autores, os campos obrigatórios que não forem preenchidos serão solicitados caso se tente gravar o registro sem preenchê-los. Também é possível saber quais os campos obrigatórios através da cor azul do nome do campo.

 

Note que neste momento este registro será exibido com o status como PUBLICADA.

 

Os procedimentos seguintes são exatamente iguais à função Incluir.
