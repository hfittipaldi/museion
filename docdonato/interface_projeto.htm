Especificação da Interface

 

A interface eleita para o projeto foi orientada para utilização em um navegador (browser).  Para tanto se definiu o layout de acordo com o modelo que foi fornecido pelo MNBA. A área definida para a interface é de 750X550 pixels, sendo a resolução de vídeo indicada de 800X600 pixels e opção de cores no mínimo em Hi Color (16 bits).

 

 

A página de abertura do sistema terá a identificação do usuário e já neste momento existirá intervenção no banco de dados, que será consultado sobre a existência deste usuário e confrontará a senha fornecida com a armazenada. Somente se o usuário existir e sua senha for validada ele terá acesso à próxima tela.

 

É também a partir desta tela que os usuários visitantes terão acesso ao sistema. Para tanto eles deverão deixar os campos de usuário e senha em branco e clicar diretamente no botão Visitantes.

 

A imagem de fundo do projeto poderá ser mudada, desde que seja substituída por uma imagem de iguais proporções, de sorte que não fiquem espaços em branco nem falhas na tela. Além disso, deve-se eleger uma imagem que se integre com os textos e elementos do projeto, de forma a não criar um impacto visual desagradável.

 

As logomarcas presentes no rodapé da página desaparecerão quando dentro do sistema. Apenas as logos da Vitae e do Museu permanecerão. A logo do Museu poderá ser substituída desde que mantidas as proporções da logo original.

 

As telas criadas estão compatíveis com os browsers Internet Explorer 6.0 e Mozilla Firefox 0.9.2. Outros browsers poderão funcionar, mas poderão apresentar comportamento diferente dos dois browsers homologados.

 

Após se validar no sistema, o usuário é passado para a tela interna do sistema, onde tem acesso aos itens de menu que sua conta tem acesso. Note que a imagem de fundo está agora diferente, sendo apenas um sombreado da imagem original. Esta imagem também deverá ser provida pelo Museu onde o sistema será executado.

 

O menu de opções ficará com seu primeiro nível fixo no lado esquerdo da tela, e estará acessível a qualquer momento da navegação.

 

 

 

Uma vez selecionada uma opção, esta mostra os sub-itens que estão disponíveis, sempre se levando em conta as permissões do usuário logado.

 

 

 

A área definida para apresentação de informações e entrada de dados está a direita da tela, entre a barra intermediária onde consta o nome do usuário logado e o fim da página, com um tamanho de 562X410 pixels. É nesta área que serão solicitadas as entradas de dados e apresentada a saída de dados. Os formulários de entrada de dados, quando necessário, terão abas que permitirão a otimização do espaço. Já as listagens terão o número de linhas por página limitadas a capacidade da área de exposição.
