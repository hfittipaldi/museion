Rotação da obra (Ficha de Catalogação)

 

O campo Imagem 3D é utilizado pelo sistema para gerar visualização da obra em rotação. Quando o campo está marcado como SIM, pode-se visualizar a imagem em rotação, tanto na ficha técnica (modo de alteração) como nas consultas.

 

Os passos para inclusão de imagens para rotação serão mostrados abaixo.

 

1.      Fotografar a obra de todos os lados (mínimo sugerido de 60 fotografias – uma para cada 6°) sobre uma superfície que se possa rodar sem movimentar a obra. Para se obter um giro completo com 72 fotografias, a base deve ser girada 5° para cada imagem (valores recomendados). No exemplo abaixo, uma rotação com 11 imagens.

 

 

2.      Salvar as imagens na seqüência certa dentro de um diretório com o número de registro da obra. No exemplo, a sugestão seria c:\escultura\3d\2800. Onde as imagens seriam armazenadas como 01.jpg, 02.jpg, etc.

3.      Se as imagens estiverem em alta definição reduza-as com um programa de edição de imagens para que a rotação não se torne excessivamente pesada. Recomendamos que cada imagem possua 50Kb aproximadamente.

4.      Abra o programa de criação de rotação (ConfigUtility.exe) que se encontra em C:\Donato3\Apache\htdocs\donato\rotate\config-utility. Dica: crie um atalho para o programa na área de trabalho.

5.      Pressione o botão “Load Images” para selecionar o diretório onde se encontram as imagens a serem carregadas, como mostrado na figura abaixo. Pressione “Ok”. As imagens existentes dentro do diretório aparecerão na lista de imagens do programa de rotação.

 

 

6.      Selecione as opções “Show Arrow Buttons” e “Show Zoom Bar” como “True”.

 

 

 

7.      Na aba “Rotation” digite “4” para o campo “Rotate Period” para suavizar a rotação. Dica: gaste algum tempo testando as possibilidades do programa para escolher as melhores configurações para você.

 

 

8.      Pressione o botão “Save”.

NOTA IMPORTANTE: o arquivo criado será no formato ZIP, e deverá ser salvo no diretório

C:\Donato3\Apache\htdocs\donato\rotate\sources

com o nome do ID da obra. O ID da obra pode ser obtido dentro da ficha técnica, na área de imagens à esquerda. No exemplo seria “4004.zip”. Ver imagem abaixo.

 

 

9.      Se o processo foi bem sucedido, o ícone de objeto 3D será visível para objetos tridimensionais na ficha de catalogação no modo de alteração e nas consultas.
