Função Acervo/Registrar Obra

Ver também Incluir – Incluir com Publicação – Alterar – Alterar com Publicação

 

Esta função permite registrar a entrada da obra no acervo do museu, incluindo apenas as informações básicas. Ela foi criada com o objetivo de agilizar o processo, permitindo que o responsável pela coleção verifique as informações incluídas para só então concluir o registro da obra e publicá-la para consultas.

 

 

Uma vez que o Registro informado esteja livre o sistema perguntará se de fato se deseja incluir esta obra. Clique em OK:

 

 

Note que o número de abas é de apenas duas, limitando o número de campos necessários para se registrar a obra.

Esta opção não permite publicar o registro.

 

 

 

Preencha todos os campos necessários para o armazenamento da obra. Além dos autores, os campos obrigatórios que não forem preenchidos serão solicitados caso se tente gravar o registro sem preenchê-los. Também é possível saber quais os campos obrigatórios através da cor azul do nome do campo.

 

Note que neste momento este registro será exibido com o status como EM CATALOGAÇÃO. Não sendo possível publicar a obra.

 

Os procedimentos seguintes são exatamente iguais à função Incluir.
