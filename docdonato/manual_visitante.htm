Utilização do Sistema - Visitantes

 

1. Introdução

 

Este documento tem como função descrever as diversas funções do Sistema Donato, mostrando ao visitante como proceder para realizar consultas no sistema.

O acesso para visitante é realizado no canto inferior direito.

 

 

2. Árvore de Menu

 

As funcionalidades do sistema Donato são acessadas a partir do menu do sistema, que pode ser visto na página principal (logo após o login no sistema, clicando no link Visitante). Abaixo reproduzimos esta página:
