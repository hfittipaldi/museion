Função Acervo/Obra/Incluir

Ver também Incluir com Publicação – Alterar – Alterar com Publicação - Registrar

 

Esta função permite incluir novas obras no acervo do museu. Para isso clique nesta opção e informe na tela a seguir o número de registro da obra:

 

 

Uma vez que o Registro informado esteja livre o sistema perguntará se de fato se deseja incluir esta obra. Clique em OK:

 

 

Devido a quantidade de campos necessários para o cadastramento da obra, foi necessário fazer este formulário utilizando-se toda a área da tela. Note que ao lado do nome do usuário logado existe um ícone que permite retornar a tela com as opções de menu.

 

 

 

Preencha todos os campos necessários para o armazenamento da obra. Os campos obrigatórios que não forem preenchidos serão solicitados caso se tente gravar o registro sem preenchê-los. Também é possível saber quais os campos obrigatórios através da cor azul do nome do campo.

 

Note que neste momento este registro será arquivado com o status como EM CATALOGAÇÃO. Isto significa que esta obra ainda não estará disponível para consultas no sistema.

 

Após cadastrar os dados básicos da obra, informe os dados complementares. Passe então para a aba Partes e informe as partes desta obra:

 

 

Clique no ícone de adicionar partes para criar as partes da obra. Informe os campos necessários e salve o registro:

 

 

Após o cadastramento da Parte será solicitado informar as suas dimensões. Informe os dados no formulário que será apresentado:

 

 

Após informar todos os dados da parte volte ao formulário da Obra:

 

 

Veja que agora temos uma parte cadastrada. Se existirem mais partes, inclua-as fazendo o mesmo procedimento.

 

Após cadastrar todas as partes clique na aba Autoria e informe os dados dos autores da obra:

 

 

Clique no ícone de adicionar autores para cadastrar os autores da obra. Informe os dados do Autor e sua hierarquia (começando de 1, se houver mais de um autor – mais importante). Selecione um autor da lista existente, ou procure por parte do nome através da busca no ícone .

 

 

A janela de busca aparecerá como mostrado na figura abaixo. No exemplo o nome “José” foi digitado e o botão Pesquisar pressionado. Note que alguns nomes na lista não têm “José”. Isto ocorre porque a busca é realizada nos campos “Nome para etiqueta” e “Nome para catálogo” do registro do autor. O sistema irá procurar em ambos, porém só irá exibir o primeiro. Após encontrar o autor pressione o botão “Confirmar”.

 

 

O nome do autor aparecerá no registro de catalogação, como mostrado abaixo

 

 

Após preencher os campos clique no botão Gravar da tela de Autores:

 

 

Refaça este procedimento para todos os autores respeitando o nível de hierarquia de cada um.

A seguir clique na aba Bibliografia e forneça os dados de bibliografia desta obra. Clique no botão de adicionar Bibliografias para criar o registro:

 

 

Procure pela referência para verificar se a mesma já se encontra cadastrada. Digite parte do título da referência bibliográfica e pressione o botão Avançar. Ao localizar a referência, pressione o ícone que se encontra à direita para adicionar à lista de referências.

 

 

Preencha os campo “Observações” com o número da página onde está citada ou reproduzida a obra, e a seguir clique no botão gravar:

 

 

Ao clicar no botão gravar será apresentada novamente a tela para inclusão de novas bibliografias:

 

 

Refaça este procedimento até ter cadastrado todas as referências bibliográficas.

A seguir clique na aba Exposição e no ícone para adicionar exposição (figura abaixo). O procedimento é o mesmo mostrado para adicionar bibliografias.

 

 

Digite parte do nome da exposição para verificar se ela já está cadastrada e pressione o botão Avançar.

 

 

No caso de uma ainda não cadastrada, será apresentada a tela para preenchimento dos dados da exposição. Note que o texto utilizado na busca na tela anterior foi copiado para o nome da exposição. O campo Observação deve ser preenchido com informações adicionais sobre a obra na exposição, no exemplo o texto “prêmio de viagem ao exterior”.

 

 

Preencha os dados apresentados na tela e clique no botão Gravar.

Já para uma exposição existente serão apresentadas as exposições encontradas que satisfaçam a condição de pesquisa pelo título informado. No exemplo “Tradição e Ruptura”:

 


Selecione a exposição desejada clicando no ícone ao lado do nome da mesma. Preencha o campo “Observações” e pressione o botão Gravar:

 

 

Depois de cadastrar todos os dados da obra, caso exista, podemos incluir a imagem da obra. Para isso volte ao menu inicial do sistema, clique na opção Acervo/Imagem/Incluir
