Primeiros passos

 

O banco de dados que você está recebendo já contém algumas informações necessárias para se iniciar uma nova base de dados, tais como: usuário ADMINISTRADOR, ESTADOS DA UNIÃO, etc.  Porém algumas alterações são necessárias para o correto funcionamento do sistema. É altamente recomendado que o responsável pelo banco de dados realize os passos seguintes:

 

1.     Incluir um novo usuário com poderes de administrador

Abra o Donato como administrador

Nome do usuário= Perfil_1

Senha = admin

Selecione ESTRUTURA – SISTEMA – USUÁRIOS no menu

Pressione o ícone  existente no final do formulário para incluir um novo usuário

Preencha a ficha com os dados do novo usuário e selecione ATIVO=SIM e STATUS=ADMINISTRADOR

Pressione o botão de GRAVAR.

Encerre a sessão do Donato pressionando o ícone  existente à direita na barra superior do programa.

Pressione o botão OK confirmando o encerramento da sessão.

 

2.     Alterar o perfil do usuário ADMINISTRADOR já existente.

Como o nome de usuário ADMINISTRADOR e sua senha são idênticos para todos os bancos de dados, é muito importante alterá-lo ou excluí-lo de seu sistema.

Selecione ESTRUTURA – SISTEMA – USUÁRIOS no menu

Pressione o ícone   para alterar o cadastro do usuário ADMINISTRADOR

Selecione ATIVO=NÃO

Pressione o botão de GRAVAR

Ou pressione o ícone  para excluir o ADMINISTRADOR.

 

3.     Incluir Coleções

Sugerimos que todas as coleções da instituição, tais como, Pintura, Escultura, Gravura, etc, sejam incluídas antes de serem inseridos os cadastros de obras. Somente as coleções já cadastradas aparecerão na lista de coleções no cadastro de obras

Para incluir uma coleção selecione ESTRUTURA – TABELAS – COLEÇÕES no menu

Pressione o ícone  existente no final do formulário para incluir uma nova coleção

Preencha todos os dados do cadastro da coleção

Pressione o botão de GRAVAR.

Repita o processo para todas as coleções de sua instituição

 

4.     Incluir autores

Os autores devem ser incluídos antes de serem inseridos os cadastros de suas obras. Somente os autores já cadastrados aparecerão na lista de autores no cadastro de obras

Selecione AUTORES – INCLUIR no menu

Digite o nome do autor e preencha o cadastro com os dados do autor.

Repita o processo para todos os autores com obras no acervo da instituição

 

5.     Incluir locais internos

Todos os locais internos da instituição que poderão armazenar obras, tais como, reservas técnicas, galerias, mapotecas, etc., deverão ser incluídos antes de serem inseridos os cadastros de obras. Somente os locais já cadastrados aparecerão na lista para futuras movimentações internas das obras.

Selecione ESTRUTURA – TABELAS – LOCAIS no menu

Pressione o ícone  existente no final do formulário para incluir um novo local interno.

Preencha o formulário com o nome do local interno e se a movimentação é controlada, tais como reserva técnica ou laboratório de restauração ou não. Os locais cadastrados como de movimentação não controlada não serão exibidos nas listas de movimentação interna.

Pressione o botão GRAVAR.

Repita o processo para todos os locais internos de sua instituição.
