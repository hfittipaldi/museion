Esta função permite incluir imagens no banco de imagens do Donato:

 

 

Preencha os dados solicitados, atentando para o Tipo de Vínculo. Para cada tipo de vínculo será apresentado o campo correspondente para vinculação:

 

 

Tipo de Vínculo

Campo a Preencher (Vinculação)

Sem Vínculo

Nenhum

Obra

Número de Registro da Obra

Autor

Nome do Autor

Instituição

Nenhum

 

Localize a imagem com o botão Procurar (ao lado da caixa de texto da imagem). Após preencher todos os dados clique no botão Gravar:

 

 

A imagem será carregada para o servidor para o diretório de imagem correspondente (informado no campo Destino da Fotografia).

 

O campo Localização é utilizado para detalhamento do local armazenado quando a opção Destino da fotografia for “Disco externo” , ver imagem abaixo.
