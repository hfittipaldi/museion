Miniatura das imagens

Ver também: Imagens

 

 

Criada a opção Miniatura no registro das imagens de obras com o objetivo de acelerar o processo das pesquisas e das impressões. Quando uma obra possui uma miniatura da imagem o sistema a exibe no lugar da imagem principal, tanto nas pesquisas como nos relatórios. Porém, se o usuário “clicar” sobre a imagem para visualizá-la no modo de consulta, a imagem que será exibida será aquela definida como Principal da obra, ou, no caso de não haver uma imagem principal, será exibida a primeira imagem inserida no sistema para aquela obra.

Sugestões para as imagens mini:

- Usar a imagem principal da obra como modelo e reduzi-la em um programa de edição de imagens para que ela tenha 150 pixels em seu maior lado, procurando reduzir ao máximo a imagem sem que ela perca muito de sua qualidade quando exibida em pequeno formato.

Para incluir uma imagem em miniatura da obra utilize a opção Acervo -> Imagens -> Incluir do menu principal. Preencha a ficha de imagem como mostrada abaixo:

Nota: sempre que a opção Mini? ¤ Sim for selecionada, as opções Principal e Laudo serão desabilitadas. O sistema aceita apenas uma imagem Mini por obra, como as imagens Principais
