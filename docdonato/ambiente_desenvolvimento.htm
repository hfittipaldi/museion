Ambiente de Desenvolvimento

 

O Projeto Donato 3.0 foi previsto de ser desenvolvido utilizando-se as ferramentas abaixo descritas:

 

  • Servidor Web Apache
  • Interpretador de Script PHP
  • Banco de Dados MySQL

 

Como sistema operacional, o MNBA optou pelo uso do Windows 2000 Server, que é aceito pelos pacotes acima especificados.

 

Para o desenvolvimento do Donato 3.0, na Visão Virtual foi configurado um servidor com estes produtos nas versões especificadas no documento “Componentes e Requisitos”.

 

Este servidor, doravante denominado “desenvolvimento”, tem as mesmas características que o servidor onde o sistema será executado deverá ter.

 

Além disso, para o tratamento de imagens, desenho das páginas, criação de formulários, modelagem dos dados, manipulação do banco de dados e controle da programação, são utilizados softwares adicionais, a saber:

 

  • Photoshop
  • Dreamweaver MX
  • ErX
  • MySQL Administrator
  • Aplicação de Controle de Desenvolvimento de Projetos (interna)

 

 Estes produtos estão instalados nos micro computadores da equipe de desenvolvimento. De acordo com a necessidade é utilizado um ou mais destes programas. Um exemplo é a programação em PHP, que fica “embebida” na forma de scripts nas páginas.

 

O servidor Apache ao encontrar uma página com terminação .php sabe que deve direciona-la para o interpretador, que irá executar os códigos dos scripts nela embutidos.

 

De acordo com o código, este pode ir ao banco de dados MySQL através de uma conexão ODBC (Open DataBase Conectivity), previamente definida no servidor, a nível do sistema operacional e manipular os dados de acordo com o comando SQL solicitado.
