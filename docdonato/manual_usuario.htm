1. Introdução

 

Este documento tem como função descrever as diversas funções do Sistema Donato, mostrando ao usuário como proceder para realizar determinada tarefa. Ao longo deste documento iremos apresentar todas as entradas de menu do sistema e as funcionalidades de cada uma.

 

É importante que o usuário que vá consultar este documento para realizar alguma tarefa no sistema já tenha se familiarizado com o documento de Convenções do Sistema, onde são apontadas as facilidades de utilização do mesmo.
