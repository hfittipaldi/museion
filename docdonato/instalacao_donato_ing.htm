Instalação do ambiente para o Donato 3.0

 

Para o perfeito funcionamento do Donato é necessário a instalação de certos pacotes de software no servidor. Estes softwares são os responsáveis pelo armazenamento das informações e apresentação das páginas no browser da estação cliente. A seguir apresentamos um  roteiro para a instalação dos softwares e os procedimentos necessários para a configuração do Windows 2000.

 

1)                Servidor Apache

 

O Servidor Apache é um web server (servidor web) desenvolvido pelo Apache Group (www.apache.com). Tem como principal característica a leveza e é adotado por mais da metade dos servidores web do mundo. Além de bastante difundido, o Apache é também um software livre, ou seja, pode-se utilizar o produto sem a necessidade de pagamento de licenças ou royalties. O Apache concorre diretamente por recursos, como por exemplo a porta de comunicação 80 do TCP/IP com o Microsoft IIS (servidor web da Microsoft), que já vem na maioria das vezes instalado e ativo no Windows 2000. Assim, antes da instalação do Apache deve-se desativar o IIS e certificar que o mesmo não reativará na próxima ativação do sistema.

 

Clique em Start-> Setting -> Control Panel

5) Carregando as páginas do Donato 3.0

 

Agora estamos prontos para a carga das páginas da aplicação. As páginas são os arquivos que serão servidos pelo Apache (web server) e processados pelo browser. Com o PHP as instruções de manipulação de banco de dados serão executadas no MySQL.

 

Para a carga das páginas no Apache siga as instruções abaixo. Abra o Windows Explorer e vá pára o CD de instalação do Donato. Localize no CD a pasta pagina. Veja a tela abaixo:

 

fig. 66 – Localizando o diretório de páginas do Donato no CD de instalação

 

Marque o sub-diretório donato clicando uma vez sobre o nome e após marcar digite CONTROL-C. Isto fará com que o diretório seja marcado para cópia. Depois, ainda no Windows Explorer vá para o diretório C:\Program Files\Apache\Apache Group\htdocs conforme a tela abaixo:

 

fig. 67 – Diretório de páginas do Apache no servidor

 

Neste diretório digite CONTROL-V. Isto fará com que o diretório Donato do CD seja copiado para baixo do diretório htdocs do Apache:

 

fig. 67 – Diretório Donato em htdocs do Apache

 

Pronto. As páginas estão copiadas. Agora vamos instalar o database do Donato no MySQL.
